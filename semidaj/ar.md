# La Araba Lingvo
*(ekzemploj el "Arabisch ohne Mühe heute" el la eldonejo Assimil)*

## (1) كتب
*(ḆṮḴ — kataba — skribis li)*

| كتب الكاتب |
|:-|
| B̥ṬḴ̱-L- ḆṮḴ |
| kataba ăl-kātibu |
| skribis la skribisto |

## (2) الكاتب
*(B̥ṬḴ̱-Lʔ̱⁠ — al-kātibu — la skribisto)*

| جلس الكاتب |
|:-|
| B̥ṬḴ̱-L- S̱ḺJ̱ |
| ĝalasa ăl-kātibu |
| sidiĝis la skribisto |

| جلس وكتب |
|:-|
| ḆṮḴ-W̱ S̱ḺJ̱ |
| ĝalasa wa-kataba |
| sidiĝis li kaj skribis |

## (3) الولد
*(D̥ḺW̱-Lʔ̱ — al-waladu — la knabo)*

| دخل الولد |
|:-|
| D̥ḺW̱-L- ḺX̱Ḏ |
| daꝁala ăl-waladu |
| eniris la knabo |

| كتب ودرس |
|:-|
| S̱ṞḎ-W̱ ḆṮḴ |
| kataba wa-darasa |
| skribis li kaj lernis |

## (4) الرجل
*(̥LJ̥Ṟ-Lʔ̱ — al-raĝulu — la viro)*

| ماذا كتب الرجل؟ |
|:-|
| ؟ ̥LJ̥Ṟ-L- ḆṮḴ Đ̱̱M̱̱ |
| māđā kataba ăl=raĝulu? |
| kion skribis la viro? |

| كتب الكتلب |
|:-|
| ḆṮ̱Ḳ-L- ḆṮḴ |
| kataba ăl-kitāba |
| skribis li la libron |

## (5) في البيت
*(ṬYḆ-L F̤ — fī ăl-bayti — en la domo)*

| دخل البيت |
|:-|
| ṮYḆ-L- ḺX̱Ḏ |
| daꝁala ăl-bayta |
| eniris li la domon |

| الرجل في البيت |
|:-|
| ṬYḆ-L- F̤ L̥J̥Ṟ Lʔ̱ |
| al-raĝulu fī ăl-bayti |
| estas la viro en la domo |

## (6) الوالد
*(D̥ḶW̱̱-Lʔ̱ — al-wālidu — la patro)*

| دخل الوالد البيت |
|:-|
| ṮYḆ-L- D̥ḶW̱̱-L- ḺX̱Ḏ |
| daꝁala ăl-wālidu ăl-bayta |
| eniris la patro la domon |

| ماذا وجد الوالد؟ |
|:-|
| ؟ D̥ḶW̱̱-L- ḎJ̱W̱ Đ̱̱M̱̱ |
| māđā waĝada ăl-wālidu? |
| kion trovis la patro? |

| وجد الكتاب |
|:-|
| ḆṮ̱Ḳ-L- ḎJ̱W̱ |
| waĝada ăl-kitāba |
| trovis li la libron |

| الوالد والولد في البيت |
|:-|
| ṬYḆ-L- F̤ D̥ḺW̱-L-W̱ D̥ḶW̱̱-Lʔ̱ |
| al-wālidu wa-ăl-waladu fī ăl-bayti |
| estas la patro kaj la knabo en la domo |

## (7)

| جلس الكاتب وكتب الكتاب |
|:-|
| ḆṮ̱Ḳ-L- ḆṮḴ-W̱ B̥ṬḴ̱-L- S̱ḺJ̱ |
| ĝalasa ăl-kātibu wa-kataba ăl-kitāba |
| sidiĝis la skribisto kaj skribis la libron |

| دخل الولد ودرس في الكتاب |
|:-|
| ḄṮ̱Ḳ-L- F̤ S̱ṞḎ-W̱ D̥ḺW̱-L- ḺX̱Ḏ |
| daꝁala ăl=waladu wa-darasa fī ăl-kitābi |
| eniris la knabo kaj lernis en la libro |

| دخل الرجل البيت ووجد الولد |
|:-|
| ḎḺW̱-L- ḎJ̱W̱-W̱ ṮYḆ-L- L̥J̥Ṟ-L- ḺX̱Ḏ |
| daꝁala ăl=raĝulu ăl-bayta wa-waĝada ăl-walada |
| eniris la viro la domon kaj trovis la knabon |

| الرجل والوالد في البيت |
|:-|
| ṬYḆ-L- F̤ D̥ḶW̱̱-L-W̱ L̥J̥Ṟ-Lʔ̱ |
| al-raĝulu wa-ăl-wālidu fī ăl-bayti |
| estas la viro kaj la patro en la domo |


## (8) البنت
*(T̥NḄ-Lʔ̱ — al-bintu — la knabino)*

| من دخل المكتب؟ |
|:-|
| ؟ ḆṮKM̱-L- ḺX̱Ḏ NM̱ |
| man daꝁala ăl-maktaba? |
| kiu eniris la skribejon? |

| دخلت البنت المكتب |
|:-|
| ḆṮKM̱-L- T̥NḄ-L̇ TḺX̱Ḏ |
| daꝁalati ăl-bintu ăl-maktaba |
| eniris la knabino la skribejon |

| جلست ودرست |
|:-|
| TS̱ṞḎ-W̱ TS̱ḺJ̱ |
| ĝalasatĭ wa-darasat |
| sidiĝis ŝi kaj lernis |

| وخرجت |
|:-|
| TJ̱ṞX̱-W̱ |
| wa-ꝁaraĝatĭ |
| kaj eliris |


## (9) بالقلم
*(ṂḺQ̱-L-Ḅ — bi-ăl-qalami — per la skribilo)*

| وجد الولد البنت وقل |
|:-|
| :ḺQ̱̱-W̱ ṮNḄ-L- D̥ḺW̱-L- ḎJ̱W̱ |
| waĝada ăl-waladu ăl-bintu wa-qāla: |
| trovis la knabo la knabinon kaj diris: |

| انا درست في الكتاب |
|:-|
| ḄṮ̱Ḳ-L- F̤ T̥SṞḎ Ṉ̱ʔ̱ |
| anā darastu fī ăl-kitābi |
| lernis mi en la libro |

| وكتبت بالقلم |
|:-|
| ṂḺQ̱-L-Ḅ T̥BṮḴ-W̱ |
| wa-katabtu bi-ăl-qalami |
| kaj skribis per la skribilo |

| القلم قلمك |
|:-|
| Ḳ-M̥ḺQ̱ M̥ḺQ̱-Lʔ̱ |
| al-qalamu qalamu-ki |
| estas la skribilo via skribilo |

## (10) قلم البنت
*(ṬNḄ-L- M̥ḺQ̱ — qalamu ăl-binti — la skribilo de la knabino)*

| الاستاذ قي المكتب |
|:-|
| ḄṮKM̱-L- F̤ Đ̥Ṯ̱Sʔ̥-Lʔ̱ |
| al-ustāđu fī ăl-maktabi |
| estas la profesoro en la skribejo |

| قال للولد |
|:-|
| :ḌḺW̱-L-Ḷ ḺQ̱̱ |
| qāla li-l-waladi: |
| diris li al la knabo: |

| انت ماذا اخذت؟ |
|:-|
| ؟ ṮĐX̱ʔ̱ Đ̱̱M̱̱ ,ṮNʔ̱ |
| anta, māđā aꝁađta? |
| vi, kion prenis vi? |

| انا اخذت قلم البنت |
|:-|
| ṬNḄ-L- M̱ḺQ̱ T̥ĐX̱ʔ̱ Ṉ̱ʔ̱ |
| anā aꝁađtu qalama ăl-binti |
| prenis mi la skribilon de la knabino |

## (11) في السوق
*(fī ăl=sūqi — en la merkato)*

| ذهب الرل إلى السوق |
|:-|
| đahaba ăl=raĝulu ilâ ăl=sūqi |
| la viro iris al la merkato |

| :قال الجد للولد |
|:-|
| qāla ăl-ĝaddu li-l-waladi: |
| la avo diris al la knabo: |

| اين والدك؟ |
|:-|
| ạna wālidu-ka? |
| estas kie via patro? |

| هو في السوق الجديد |
|:-|
| huwa fī ăl=sūqi ăl-ĝadīdi |
| estas li en la nova merkato |

## (12) التلميذ
*(al-tilmīđu — la lernanto)*

| ما درس التلميذ درسه |
|:-|
| mā darasa ăl=tilmīđu darsa-hu |
| la lernanto ne lernis sian lernaĵon |

| :قال الاستاذ للتلميذ |
|:-|
| qāla ăl-ustāđu li=l-tilmīđi: |
| diris la profesoro: |

| !ما درست درسك |
|:-|
| mā darasta darsa-ka! |
| ne lernis vi vian lernaĵon! |

| وخرج من المكتب |
|:-|
| wa-ꝁaraĝa mina ăl-maktabi |
| kaj eliris li el la skribejo |

## (13) قط البنت
*(qiţţu ăl-binti — la kato de la knabino)*

| انت هل وخدت قطك؟ |
|:-|
| anti hal waĝadti qiţţa-ki? |
| ĉu trovis vi vian katon? |

| لا! ما وجدت قطي |
|:-|
| la! mā waꝁadtu qiţţă-ī |
| ne! ne trovis mi mian katon |

| القط الجميل |
|:-|
| al-qiţţu ăl-ĝamīlu |
| estas la bela kato |

| في وسط البستان |
|:-|
| fī wasaţi ăl-bustāni |
| en la mezo de la ĝardeno |
