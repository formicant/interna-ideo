# La Araba Lingvo
*(ekzemploj el "Arabisch ohne Mühe heute" el la eldonejo Assimil)*

## (1) كتب
*(ḆṮḴ — kataba — li skribis)*

| كتب الكاتب |
|:-|
| B̥ṬḴ̱-L- ḆṮḴ |
| kataba ăl-kātibu |
| skribis la skribisto |

## (2) الكاتب
*(B̥ṬḴ̱-Lʔ̱⁠ — al-kātibu — la skribisto)*

| جلس الكاتب |
|:-|
| B̥ṬḴ̱-L- S̱ḺJ̱ |
| jalasa ăl-kātibu |
| sidiĝis la skribisto |

| جلس وكتب |
|:-|
| ḆṮḴ-W̱ S̱ḺJ̱ |
| jalasa wa-kataba |
| li sidiĝis kaj skribis |

## (3) الولد
*(D̥ḺW̱-Lʔ̱ — al-waladu — la knabo)*

| دخل الولد |
|:-|
| D̥ḺW̱-L- ḺX̱Ḏ |
| daxala ăl-waladu |
| eniris la knabo |

| كتب ودرس |
|:-|
| S̱ṞḎ-W̱ ḆṮḴ |
| kataba wa-darasa |
| li skribis kaj lernis |

## (4) الرجل
*(̥LJ̥Ṟ-Lʔ̱ — al-rajulu — la viro)*

| ماذا كتب الرجل؟ |
|:-|
| ؟ ̥LJ̥Ṟ-L- ḆṮḴ Đ̱̱M̱̱ |
| māđā kataba ăl=rajulu? |
| kion skribis la viro? |

| كتب الكتلب |
|:-|
| ḆṮ̱Ḳ-L- ḆṮḴ |
| kataba ăl-kitāba |
| li skribis la libron |

## (5) في البيت
*(ṬYḆ-L F̤ — fī ăl-bayti — en la domo)*

| دخل البيت |
|:-|
| ṮYḆ-L- ḺX̱Ḏ |
| daxala ăl-bayta |
| li eniris la domon |

| الرجل في البيت |
|:-|
| ṬYḆ-L- F̤ L̥J̥Ṟ Lʔ̱ |
| al-rajulu fī ăl-bayti |
| la viro estas en la domo |

## (6) الوالد
*(D̥ḶW̱̱-Lʔ̱ — al-wālidu — la patro)*

| دخل الوالد البيت |
|:-|
| ṮYḆ-L- D̥ḶW̱̱-L- ḺX̱Ḏ |
| daxala ăl-wālidu ăl-bayta |
| eniris la patro la domon |

| ماذا وجد الوالد؟ |
|:-|
| ؟ D̥ḶW̱̱-L- ḎJ̱W̱ Đ̱̱M̱̱ |
| māđā wajada ăl-wālidu? |
| kion trovis la patro? |

| وجد الكتاب |
|:-|
| ḆṮ̱Ḳ-L- ḎJ̱W̱ |
| wajada ăl-kitāba |
| li trovis la libron |

| الوالد والولد في البيت |
|:-|
| ṬYḆ-L- F̤ D̥ḺW̱-L-W̱ D̥ḶW̱̱-Lʔ̱ |
| al-wālidu wa-ăl-waladu fī ăl-bayti |
| la patro kaj la knabo estas en la domo |

## (7)

| جلس الكاتب وكتب الكتاب |
|:-|
| ḆṮ̱Ḳ-L- ḆṮḴ-W̱ B̥ṬḴ̱-L- S̱ḺJ̱ |
| jalasa ăl-kātibu wa-kataba ăl-kitāba |
| sidiĝis la skribisto kaj skribis la libron |

| دخل الولد ودرس في الكتاب |
|:-|
| ḄṮ̱Ḳ-L- F̤ S̱ṞḎ-W̱ D̥ḺW̱-L- ḺX̱Ḏ |
| daxala ăl=waladu wa-darasa fī ăl-kitābi |
| eniris la knabo kaj lernis en la libro |

| دخل الرجل البيت ووجد الولد |
|:-|
| ḎḺW̱-L- ḎJ̱W̱-W̱ ṮYḆ-L- L̥J̥Ṟ-L- ḺX̱Ḏ |
| daxala ăl=rajulu ăl-bayta wa-wajada ăl-walada |
| eniris la viro la domon kaj trovis la knabon |

| الرجل والوالد في البيت |
|:-|
| ṬYḆ-L- F̤ D̥ḶW̱̱-L-W̱ L̥J̥Ṟ-Lʔ̱ |
| al-rajulu wa-ăl-wālidu fī ăl-bayti |
| la viro kaj la patro estas en la domo |


## (8) البنت
*(T̥NḄ-Lʔ̱ — al-bintu — la knabino)*

| من دخل المكتب؟ |
|:-|
| ؟ ḆṮKM̱-L- ḺX̱Ḏ NM̱ |
| man daxala ăl-maktaba? |
| kiu eniris la skribejon? |

| دخلت البنت المكتب |
|:-|
| ḆṮKM̱-L- T̥NḄ-L̇ TḺX̱Ḏ |
| daxalati ăl-bintu ăl-maktaba |
| eniris la knabino la skribejon |

| جلست ودرست |
|:-|
| TS̱ṞḎ-W̱ TS̱ḺJ̱ |
| jalasatĭ wa-darasat |
| ŝi sidiĝis kaj lernis |

| وخرجت |
|:-|
| TJ̱ṞX̱-W̱ |
| wa-xarajatĭ |
| kaj ŝi eliris |


## (9) بالقلم
*(ṂḺQ̱-L-Ḅ — bi-ăl-qalami — per la skribilo)*

| وجد الولد البنت وقل |
|:-|
| :ḺQ̱̱-W̱ ṮNḄ-L- D̥ḺW̱-L- ḎJ̱W̱ |
| wajada ăl-waladu ăl-bintu wa-qāla: |
| trovis la knabo la knabinon kaj diris: |

| انا درست في الكتاب |
|:-|
| ḄṮ̱Ḳ-L- F̤ T̥SṞḎ Ṉ̱ʔ̱ |
| anā darastu fī ăl-kitābi |
| mi lernis en la libro |

| وكتبت بالقلم |
|:-|
| ṂḺQ̱-L-Ḅ T̥BṮḴ-W̱ |
| wa-katabtu bi-ăl-qalami |
| kaj mi skribis per la skribilo |

| القلم قلمك |
|:-|
| Ḳ-M̥ḺQ̱ M̥ḺQ̱-Lʔ̱ |
| al-qalamu qalamu-ki |
| la skribilo estas via skribilo |

## (10) قلم البنت
*(ṬNḄ-L- M̥ḺQ̱ — qalamu ăl-binti — la skribilo de la knabino)*

| الاستاذ قي المكتب |
|:-|
| ḄṮKM̱-L- F̤ Đ̥Ṯ̱Sʔ̥-Lʔ̱ |
| al-ustāđu fī ăl-maktabi |
| la profesoro estas en la skribejo |

| قال للولد |
|:-|
| :ḌḺW̱-L-Ḷ ḺQ̱̱ |
| qāla li-l-waladi: |
| li diris al la knabo: |

| انت ماذا اخذت؟ |
|:-|
| ؟ ṮĐX̱ʔ̱ Đ̱̱M̱̱ ,ṮNʔ̱ |
| anta, māđā axađta? |
| vi, kion vi prenis? |

| انا اخذت قلم البنت |
|:-|
| ṬNḄ-L- M̱ḺQ̱ T̥ĐX̱ʔ̱ Ṉ̱ʔ̱ |
| anā axađtu qalama ăl-binti |
| mi prenis la skribilon de la knabino |

## (11) في السوق
*(fī ăl=sūqi — en la merkato)*

| ذهب الرل إلى السوق |
|:-|
| đahaba ăl=rajulu ilâ ăl=sūqi |
| la viro iris al la merkato |

| :قال الجد للولد |
|:-|
| qāla ăl-jaddu li-l-waladi: |
| la avo diris al la knabo: |

| اين والدك؟ |
|:-|
| ạna wālidu-ka? |
| kie estas via patro? |

| هو في السوق الجديد |
|:-|
| huwa fī ăl=sūqi ăl-jadīdi |
| li estas en la nova merkato |

## (12) التلميذ
*(al-tilmīđu — la lernanto)*

| ما درس التلميذ درسه |
|:-|
| mā darasa ăl=tilmīđu darsa-hu |
| la lernanto ne lernis sian lernaĵon |

| :قال الاستاذ للتلميذ |
|:-|
| qāla ăl-ustāđu li=l-tilmīđi: |
| diris la profesoro: |

| !ما درست درسك |
|:-|
| mā darasta darsa-ka! |
| vi ne lernis vian lernaĵon! |

| وخرج من المكتب |
|:-|
| wa-xaraja mina ăl-maktabi |
| kaj li eliris el la skribejo |
