# La Araba Lingvo
*(ekzemploj el "Arabisch ohne Mühe heute" el la eldonejo Assimil)*

## (1) كتب
*(kataba — skribis li)*

| كتب الكاتب |
|:-|
| kataba ăl-kātibu |
| skribis la skribisto |

## (2) الكاتب
*(al-kātibu — la skribisto)*

| جلس الكاتب |
|:-|
| ĝalasa ăl-kātibu |
| sidiĝis la skribisto |

| جلس وكتب |
|:-|
| ĝalasa wa-kataba |
| sidiĝis li kaj skribis |

## (3) الولد
*(al-waladu — la knabo)*

| دخل الولد |
|:-|
| daꝁala ăl-waladu |
| eniris la knabo |

| كتب ودرس |
|:-|
| kataba wa-darasa |
| skribis li kaj lernis |

## (4) الرجل
*(al-raĝulu — la viro)*

| ماذا كتب الرجل؟ |
|:-|
| māđā kataba ăl=raĝulu? |
| kion skribis la viro? |

| كتب الكتلب |
|:-|
| kataba ăl-kitāba |
| skribis li la libron |

## (5) في البيت
*(fī ăl-bayti — en la domo)*

| دخل البيت |
|:-|
| daꝁala ăl-bayta |
| eniris li la domon |

| الرجل في البيت |
|:-|
| al-raĝulu fī ăl-bayti |
| estas la viro en la domo |

## (6) الوالد
*(al-wālidu — la patro)*

| دخل الوالد البيت |
|:-|
| daꝁala ăl-wālidu ăl-bayta |
| eniris la patro la domon |

| ماذا وجد الوالد؟ |
|:-|
| māđā waĝada ăl-wālidu? |
| kion trovis la patro? |

| وجد الكتاب |
|:-|
| waĝada ăl-kitāba |
| trovis li la libron |

| الوالد والولد في البيت |
|:-|
| al-wālidu wa-ăl-waladu fī ăl-bayti |
| estas la patro kaj la knabo en la domo |

## (7)

| جلس الكاتب وكتب الكتاب |
|:-|
| ĝalasa ăl-kātibu wa-kataba ăl-kitāba |
| sidiĝis la skribisto kaj skribis la libron |

| دخل الولد ودرس في الكتاب |
|:-|
| daꝁala ăl=waladu wa-darasa fī ăl-kitābi |
| eniris la knabo kaj lernis en la libro |

| دخل الرجل البيت ووجد الولد |
|:-|
| daꝁala ăl=raĝulu ăl-bayta wa-waĝada ăl-walada |
| eniris la viro la domon kaj trovis la knabon |

| الرجل والوالد في البيت |
|:-|
| al-raĝulu wa-ăl-wālidu fī ăl-bayti |
| estas la viro kaj la patro en la domo |


## (8) البنت
*(al-bintu — la knabino)*

| من دخل المكتب؟ |
|:-|
| man daꝁala ăl-maktaba? |
| kiu eniris la skribejon? |

| دخلت البنت المكتب |
|:-|
| daꝁalati ăl-bintu ăl-maktaba |
| eniris la knabino la skribejon |

| جلست ودرست |
|:-|
| ĝalasatĭ wa-darasat |
| sidiĝis ŝi kaj lernis |

| وخرجت |
|:-|
| wa-ꝁaraĝatĭ |
| kaj eliris |


## (9) بالقلم
*(bi-ăl-ķalami — per la skribilo)*

| وجد الولد البنت وقل |
|:-|
| waĝada ăl-waladu ăl-binta wa-ķāla: |
| trovis la knabo la knabinon kaj diris: |

| انا درست في الكتاب |
|:-|
| anā darastu fī ăl-kitābi |
| lernis mi en la libro |

| وكتبت بالقلم |
|:-|
| wa-katabtu bi-ăl-ķalami |
| kaj skribis per la skribilo |

| القلم قلمك |
|:-|
| al-ķalamu ķalamu-ki |
| estas la skribilo via skribilo |

## (10) قلم البنت
*(ķalamu ăl-binti — la skribilo de la knabino)*

| الاستاذ قي المكتب |
|:-|
| al-ustāđu fī ăl-maktabi |
| estas la profesoro en la skribejo |

| قال للولد |
|:-|
| ķāla li-l-waladi: |
| diris li al la knabo: |

| انت ماذا اخذت؟ |
|:-|
| anta, māđā aꝁađta? |
| vi, kion prenis vi? |

| انا اخذت قلم البنت |
|:-|
| anā aꝁađtu ķalama ăl-binti |
| prenis mi la skribilon de la knabino |

## (11) في السوق
*(fī ăl=sūķi — en la merkato)*

| ذهب الرل إلى السوق |
|:-|
| đahaba ăl=raĝulu ilâ ăl=sūķi |
| la viro iris al la merkato |

| :قال الجد للولد |
|:-|
| ķāla ăl-ĝaddu li-l-waladi: |
| la avo diris al la knabo: |

| اين والدك؟ |
|:-|
| ạna wālidu-ka? |
| estas kie via patro? |

| هو في السوق الجديد |
|:-|
| huwa fī ăl=sūķi ăl-ĝadīdi |
| estas li en la nova merkato |

## (12) التلميذ
*(al-tilmīđu — la lernanto)*

| ما درس التلميذ درسه |
|:-|
| mā darasa ăl=tilmīđu darsa-hu |
| la lernanto ne lernis sian lernaĵon |

| :قال الاستاذ للتلميذ |
|:-|
| ķāla ăl-ustāđu li=l-tilmīđi: |
| diris la profesoro: |

| !ما درست درسك |
|:-|
| mā darasta darsa-ka! |
| ne lernis vi vian lernaĵon! |

| وخرج من المكتب |
|:-|
| wa-ꝁaraĝa mina ăl-maktabi |
| kaj eliris li el la skribejo |

## (13) قط البنت
*(ķițțu ăl-binti — la kato de la knabino)*

| انت هل وخدت قطك؟ |
|:-|
| anti hal waĝadti ķițța-ki? |
| ĉu trovis vi vian katon? |

| لا! ما وجدت قطي |
|:-|
| la! mā waꝁadtu ķițță-ī |
| ne! ne trovis mi mian katon |

| القط الجميل |
|:-|
| al-ķițțu ăl-ĝamīlu |
| estas la bela kato |

| في وسط البستان |
|:-|
| fī wasați ăl-bustāni |
| en la mezo de la ĝardeno |
